# Flat Surface Cleaner

Add-on do Blender para planarizar seleções e recriar uma única face limpa.

## Idioma
- Padrão: **Português**.
- Para alternar, abra `Edit > Preferences... > Add-ons`, procure por **Flat Surface Cleaner** e, no painel **Addon Preferences**, escolha o idioma em **Language/Idioma**.
- Labels, descrições e mensagens de aviso/erro respeitam a preferência selecionada.
